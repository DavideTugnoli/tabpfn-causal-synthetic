# CausalDiffTab Custom SCM Results

This final lightweight bundle contains the 500 validated CausalDiffTab result
rows for Custom SCM: 100 paired seeds at each training size in
`20,50,100,200,500`.

The seed sets exactly match the canonical cleaned TabPFN Custom SCM seed sets.
The public cleaned CSV excludes machine-specific paths and debug fields.
Additional summary tables compare CausalDiffTab with the paired TabPFN runs.
