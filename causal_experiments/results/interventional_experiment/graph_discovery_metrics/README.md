# Graph Discovery Metrics

This folder contains tooling and artifacts for graph discovery diagnostics used in
interventional-experiment analyses.

Scripts:
- `scripts/compute_discovery_metrics.py` (PC/CPDAG discovery metrics)

Default output folder:
- `scripts/outputs/cpdag_discovery_metrics/`

Generated files:
- `cpdag_discovery_metrics/discovery_metrics_per_run.csv`
- `cpdag_discovery_metrics/discovery_metrics_summary.csv`
